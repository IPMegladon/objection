Command: android hooking watch class

Usage: android hooking watch class <class>[!method] [--dump-args] [--dump-backtrace] [--dump-return]

Hooks a specified class' methods and reports on invocations.

Flags:
   --dump-args       Dump method arguments when invoked
   --dump-backtrace  Dump the call stack backtrace
   --dump-return     Dump method return values

Examples:
   android hooking watch class com.example.test 
   android hooking watch class com.example.test --dump-args --dump-return 
   android hooking watch class com.example.test!onCreate
